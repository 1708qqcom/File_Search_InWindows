# File_Search_InWindows
在windows系统指定文件夹路径下，查询系统中相同的文件夹名和文件名，并且打印出来

# 使用方法
1.运行py文件
```
python file.py
```
2.或者直接运行exe文件，File_exe的file.exe文件，一样的效果
